# ⚔️ Java 文字格斗游戏 (Text Fighting Game)

这是一个基于 Java 控制台实现的文字格斗游戏。玩家可以进行注册登录，创建属于自己的英雄角色，并通过回合制战斗不断挑战越来越强的敌人，获取道具和属性。

## ✨ 功能特性

*   **用户系统**：支持用户注册、登录及找回密码功能。
*   **角色创建**：自定义分配属性点（生命、攻击、防御）打造专属英雄。
*   **回合制战斗**：
    *   普通攻击与技能系统（强力一击、生命汲取）。
    *   怒气/魔法值消耗机制。
    *   随机掉落道具系统。
*   **成长体系**：每胜利3场战斗，角色属性自动提升，敌人也会随之强化。
*   **防作弊机制**：登录验证码、密码强度校验、账号锁定机制。

## 🛠️ 技术栈

*   **语言**：Java (JDK 8+)
*   **核心概念**：面向对象 (OOP)、枚举 (Enum)、集合 (ArrayList)、异常处理
*   **架构**：MVC 思想（Service 处理逻辑，Ui 处理视图，Control 控制流程）

## 🚀 如何运行

1.  克隆本项目到本地：
git clone https://github.com/TeioCross/java-fightgame.git
3.  使用 IntelliJ IDEA (或 Eclipse) 打开项目。
4.  找到 `app.java` 文件。
5.  运行 `main` 方法即可开始游戏。

## 🎮 操作指南

1.  启动程序后，根据菜单选择 **注册(2)** 创建一个新账号。
2.  注册完成后，选择 **登录(1)**。
3.  输入账号密码及验证码进入游戏。
4.  创建角色时，共有 20 点属性点供自由分配。
5.  进入战斗后：
*   选择 `1`: 普通攻击
*   选择 `2`: 强力一击 (消耗 HP/MP)
*   选择 `3`: 生命汲取 (消耗 HP/MP)
*   选择 `4`: 使用背包道具
6.  战斗胜利后可选择继续挑战或退出。

├── app.java          # 程序入口

├── Service.java      # 业务逻辑处理 (工具类)

├── Ui.java           # 用户界面展示

├── Users.java        # 用户实体类

├── Hero.java         # 英雄实体类

├── enemy.java        # 敌人枚举类

├── Skill.java        # 技能实体类

├── Item.java         # 道具枚举类

└── YZM.java          # 验证码工具类
